# 2026tripplan

鳥取・島根 + 天橋立 3泊4日 家族モデルプラン（2026/8/17〜8/20）

家族4人（おとな2＋子ども2）で楽しむ、モバイル前提の1枚もの旅程ページ `index.html` です。

- **4日間のタイムライン**：天橋立 → 伊根の舟屋 → 城崎温泉 → 鳥取砂丘 → 三朝温泉 → コナンの里 → 松江 → 玉造温泉 → 出雲大社
- **移動時間つき**：各区間の所要時間と、スポットごとの Google マップリンク
- **夏の時期別予算**：夏の平日／夏の週末／お盆の3段階をタブ切替で算出（宿は中〜上級グレードの温泉宿2食付が基準）
- **夏の旅のコツ**：早割・乗り捨て予約・砂丘の暑さ対策・海遊びなど
- **予約タイムライン**：宿・航空券・レンタカー・体験などを「いつ予約すべきか」を時期別に整理

`index.html` をブラウザで開くだけで閲覧できます。
