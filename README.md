鳥取・島根 3泊4日 家族モデルプラン（2026/8/16夜発〜8/20）

行きは寝台特急サンライズ出雲、帰りは飛行機（鳥取→羽田）で、山陰を西から東へ巡るプランです。モバイル前提の1枚もの旅程ページ `index.html`。

- **行き**：サンライズ出雲（東京 8/16夜発 → 出雲市 8/17朝着）
- **4日間のタイムライン**：出雲大社 → 玉造温泉 → 松江城 → 倉吉 → 三朝温泉 → コナンの里 → 鳥取砂丘
- **帰り**：鳥取砂丘コナン空港 → 羽田（米子鬼太郎空港でも可）
- **予算**：この日程に合わせた固定見積り（合計目安 約43.5万〜65.5万）
- **予約タイムライン**：サンライズ寝台（7/16 10時発売）を筆頭に、宿・航空券・体験を時期別に整理
