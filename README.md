# 2026tripplan

四国・家族旅行プラン 4泊5日（2026/8/15夜発〜8/19）

行きは寝台特急サンライズ瀬戸、帰りは新幹線（岡山経由）。高松を拠点に島めぐり＋祖谷温泉1泊の家族プラン（夫婦・13歳・8歳）。モバイル前提の1枚もの旅程ページ `index.html`。

- **行き**：サンライズ瀬戸（東京 8/15夜発 → 高松 8/16朝着）／取れない場合のB案（新幹線）も掲載
- **行程**：直島（地中美術館）→ 小豆島（エンジェルロード・寒霞渓）→ 祖谷・大歩危 → 栗林公園
- **宿候補**：高松（クレメント高松ほか）／祖谷温泉（ホテル祖谷温泉ほか）
- **概算費用**：A案ベースで総額約48万（予算50万以内）
- **予約チェックリスト**：サンライズ瀬戸（7/15 10時打ち）・地中美術館・レンタカー・宿・潮見表

各スポットに Google マップリンク付き。`index.html` をブラウザで開くだけで閲覧できます。

> 注：元資料の曜日は1日ずれていた（2025年基準）ため、2026年の正しい曜日 8/15(土)〜19(水) に修正しています。
