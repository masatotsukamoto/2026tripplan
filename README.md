# 2026tripplan

鳥取・島根 + 天橋立 3泊4日 家族モデルプラン（改訂版）

元の観光プラン画像を精査し、以下を反映した1枚もの（モバイル前提）の旅程ページ `index.html` です。

- **初日の長距離運転を解消**：天橋立観光のあとは城崎温泉泊にして到着を早めた
- **宿を3か所に分散**：城崎 → 三朝 → 玉造（松江しんじ湖）の湯めぐり構成
- **3日目の動線を最短化**：倉吉 → コナンの里 → 松江 を一直線に
- **季節別の予算**：閑散期／通常期／繁忙期の3段階で具体化（タブ切替で合計が変わる）

各スポットに Google マップへのリンク付き。`index.html` をブラウザで開くだけで閲覧できます。
